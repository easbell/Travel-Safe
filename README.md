# TravelSafe

#### Compiled travel information for advisory levels and recommended vaccines.

## Set-Up
- `clone` this repo
- `clone` [back-end repo](https://github.com/easbell/travel-safe-api/)
- `cd` into repo
- `npm install`
- `npm start`

## Tech Stack
- React / Redux
- Router
- Jest / Enzyme
- Thunks
- Node.js / Express on backend
